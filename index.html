Гайд по CSS селекторам

Типи селекторів:

  • ID селектор - "найсильніший" тип селекторів, застосовується до елементу з вказаним унікальним id
  • Селектор класу - самий використовуваний і універсальний тип селектора, застосовується до всіх елементів з відповідним класом
  • Тег селектор - засто до всіх відповідниї тегів (не завжди являється хорошим вибором селектору)
  • Селектор атрибуту - застосовується до елементів з відповідним атрибутом

Події фільму розгортаються після показаних у епізоді VIII «Зоряні війни. Останні Джедаї». Фільм завершує неймовірну історію сім’ї Скайвокеров, яка триває вже більше ніж 40 років. Стрічка обіцяє дати відповіді на всі загадки з попередніх серій. На вас чекатимуть старі та нові герої, унікальні світи та незабутні мандрівки по краю Галактики. Це буде справжній грандіозний фінал фантастичної саги!
