Гайд по CSS селекторам

Типи селекторів:

  • ID селектор - "найсильніший" тип селекторів, застосовується до елементу з вказаним унікальним id
  • Селектор класу - самий використовуваний і універсальний тип селектора, застосовується до всіх елементів з відповідним класом
  • Тег селектор - засто до всіх відповідниї тегів (не завжди являється хорошим вибором селектору)
  • Селектор атрибуту - застосовується до елементів з відповідним атрибутом
